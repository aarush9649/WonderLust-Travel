# WonderLust-Travel
Wonderlust is a full-stack travel and stay booking web application inspired by modern accommodation platforms. Users can explore destinations, view property listings, and book stays with a smooth and user-friendly experience.

## 🚀 Features

- User Registration & Login
- Secure Authentication
- Add, Edit, Delete Listings
- Browse Properties / Destinations
- Booking System
- Search and Filter Listings
- Responsive Design
- User-Friendly Interface

## 🛠️ Tech Stack

### Frontend:
- HTML
- CSS

- Bootstrap / Tailwind CSS
- React.js (if used)

### Backend:
